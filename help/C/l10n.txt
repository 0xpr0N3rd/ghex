SCREENSHOT INSTRUCTIONS
------------------------
ghex_window_empty.png

1. Start GHex (Applications -> Accessories -> Hex Editor). 
2. Ensure that the window has focus. If it does not, click on the window to give it focus.
3. Take a screenshot of the empty window exactly as it is when you start the application. 
4. Save the file to PNG.

ghex_window_open.png

1. Choose File -> Open to open a file in GHex.
2. Open any file in the file system.
3. Ensure that the window has focus.
4. Take a screenshot of the GHex window with a file open.
5. Save the file to PNG.
